OOMWOO is an open-source home robot vacuum. Code under Apache-2.0; hardware design files under an open hardware license (TBD). Sponsored by makerspet.com & remake.ai — a kit will be optional; everything here stays open. The name is a rotational ambigram — it reads the same flipped 180°, like the robot roaming your floor.
